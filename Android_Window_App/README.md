# Android APK для управления окнами

Это минимальное Android-приложение-обертка. Оно открывает мобильный интерфейс ESP32:

```text
http://192.168.100.5/mobile
```

Внутри интерфейса есть выбор окна/RP2040 и кнопки:

- `Открыто`
- `Закрыто`
- `Проветривание`
- `Стоп`

Настройки соединения задаются в самом мобильном интерфейсе: адрес ESP32, логин и пароль.

## Как собрать APK

1. Откройте папку `Android_Window_App` в Android Studio.
2. Дождитесь синхронизации Gradle.
3. Выберите `Build -> Build Bundle(s) / APK(s) -> Build APK(s)`.
4. APK появится в:

```text
app/build/outputs/apk/debug/app-debug.apk
```

## Важно

Приложение разрешает обычный HTTP, потому что ESP32 работает по локальному адресу вида `http://192.168...`.
