# Версия 2026-07-08 v1.4 web-update

Эта версия добавляет web-обновление прошивки ESP32.

## Что входит

- `ESP32_CC1101_RS485_Master_v1.4.ino` - исходник ESP32.
- `ESP32_CC1101_RS485_Master_v1.4.bin` - основной бинарный файл ESP32 для web-обновления.
- `ESP32_CC1101_RS485_Master_v1.4.merged.bin` - полный образ ESP32 для первичной прошивки.

## Изменения

- В `/config` добавлен раздел `Обновление прошивки`.
- ESP32 можно обновлять через web-интерфейс файлом `.bin`.
- После успешной загрузки ESP32 автоматически перезагружается.
- В интерфейс добавлено предупреждение по RP2040: для удаленного web-обновления RP2040 нужен отдельный bootloader.

## Важно

Через web-интерфейс ESP32 нужно загружать файл:

```text
ESP32_CC1101_RS485_Master_v1.4.bin
```

Файл `merged.bin` через web-обновление не загружать. Он нужен для полной первичной прошивки через USB/serial.

Удаленное обновление RP2040 в этой версии не включено, потому что текущая стабильная RP2040-прошивка не содержит безопасного загрузчика с резервным образом.
