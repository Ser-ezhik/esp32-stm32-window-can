# Версия 2026-07-08 v1.3 watchdog-counter

Эта версия добавляет диагностику watchdog-перезапусков RP2040.

## Что входит

- `ESP32_CC1101_RS485_Master_v1.3.ino` - исходник ESP32.
- `ESP32_CC1101_RS485_Master_v1.3.bin` - основной бинарный файл ESP32.
- `ESP32_CC1101_RS485_Master_v1.3.merged.bin` - полный образ ESP32.
- `RP2040_RS485_Window_Node_4A_v1.3.ino` - исходник RP2040.
- `RP2040_RS485_Window_Node_4A_v1.3.uf2` - прошивка RP2040.

## Изменения

- RP2040 считает перезапуски, вызванные watchdog.
- Счетчик хранится во flash RP2040 и не сбрасывается при обычном отключении питания.
- В статус RP2040 добавлено поле `watchdogResets`.
- В веб-интерфейсе ESP32 добавлена строка `Watchdog reset`.
- Добавлена кнопка сброса счетчика для локальных RP2040 и RP2040 по RS-485.

Счетчик увеличивается только если RP2040 перезагрузилась именно по watchdog. Обычная перезагрузка питанием счетчик не увеличивает.
