# Версия 2026-07-08 v1.2 reed-labels

Эта версия обновляет веб-интерфейс ESP32.

## Что входит

- `ESP32_CC1101_RS485_Master_v1.2.ino` - исходник ESP32.
- `ESP32_CC1101_RS485_Master_v1.2.bin` - основной бинарный файл ESP32.
- `ESP32_CC1101_RS485_Master_v1.2.merged.bin` - полный образ ESP32.

## Изменения

В веб-интерфейсе вместо строки вида:

```text
Герконы: 0, 0, 0, 0, 0
```

теперь отображается:

```text
Герконы: Открыто: 0 | Проветривание: 0 | Закрыто: 0 | Доп. 1: 0 | Доп. 2: 0
```

Изменение применяется для локальных RP2040 по UART и для RP2040 по RS-485.
