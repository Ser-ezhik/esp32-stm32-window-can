# Версия 2026-07-08 v1.2 reed-stop

Эта версия основана на `v1.1-safety-fixes` и меняет логику герконов в RP2040.

## Что входит

- `RP2040_RS485_Window_Node_4A_v1.2.ino` - исходник RP2040.
- `RP2040_RS485_Window_Node_4A_v1.2.uf2` - готовая прошивка RP2040.

## Изменения относительно v1.1

- `GP16` (`Открыто`) теперь останавливает все актуаторы при выполнении команды `Открыто`.
- `GP17` (`Проветривание`) теперь останавливает все актуаторы при выполнении команды `Проветривание`.
- `GP18` (`Закрыто`) теперь останавливает все актуаторы при выполнении команды `Закрыто`.
- Повторное нажатие команды уже достигнутого положения не запускает актуаторы.

## Распиновка герконов

- `GP16` - положение `Открыто`.
- `GP17` - положение `Проветривание`.
- `GP18` - положение `Закрыто`.
- `GP19` - дополнительный вход, отображается в статусе.
- `GP22` - дополнительный вход, отображается в статусе.

Герконы активны уровнем `3.3 В` (`лог. 1`), входы RP2040 не являются 5-вольтовыми.
