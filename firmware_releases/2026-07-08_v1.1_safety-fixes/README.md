# Версия 2026-07-08 v1.1 safety-fixes

Эта папка содержит сохраненную версию прошивок ESP32 и RP2040 после исправлений надежности.

## Что входит

- `ESP32_CC1101_RS485_Master_v1.1.ino` - исходник ESP32.
- `ESP32_CC1101_RS485_Master_v1.1.bin` - основной бинарный файл ESP32 для OTA/прошивки приложения.
- `ESP32_CC1101_RS485_Master_v1.1.merged.bin` - полный образ ESP32.
- `RP2040_RS485_Window_Node_4A_v1.1.ino` - исходник RP2040.
- `RP2040_RS485_Window_Node_4A_v1.1.uf2` - прошивка RP2040 для копирования на диск платы.

## Изменения

- Добавлена версия прошивки `v1.1-safety-fixes` в статусы ESP32 и RP2040.
- ESP32: защищен RF-буфер пульта через `portMUX_TYPE`.
- ESP32: добавлена безопасная инициализация записей кнопок пультов.
- ESP32: усилена обработка импорта настроек, добавлена защита от слишком большого backup-файла.
- RP2040: добавлен watchdog.
- RP2040: I2C снижен до 100 kHz для повышения устойчивости.
- RP2040: добавлена повторная инициализация INA219.
- RP2040: CAP1188 проверяется до включения реле.
- RP2040: повторная команда уже достигнутого положения не запускает актуаторы.
- RP2040: JSON-статус экранирует строки.
- RP2040: буфер flash-сохранения перенес из стека в static-память.
- RP2040: watchdog обновляется перед и после записи настроек во flash.

## Сборка

ESP32:

```powershell
arduino-cli compile --fqbn esp32:esp32:esp32s3 --export-binaries ArduinoIDE/ESP32_CC1101_RS485_Master
```

RP2040:

```powershell
arduino-cli compile --fqbn arduino:mbed_rp2040:pico --export-binaries ArduinoIDE/RP2040_RS485_Window_Node_4A
```
