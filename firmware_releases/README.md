# Архив версий прошивок

В этой папке сохраняются готовые версии прошивок ESP32 и RP2040.

Правило работы:

- рабочие `.ino` в `ArduinoIDE/` можно редактировать;
- готовые проверенные версии копируются сюда в отдельную папку;
- имена папок содержат дату, номер версии и краткое описание;
- старые папки релизов не затираются, чтобы можно было вернуться назад.

## Текущие версии

- `2026-07-08_v1.1_safety-fixes` - версия с исправлениями надежности RF, backup, watchdog, INA219, CAP1188 и I2C.

## Как выбирать файл

- Для ESP32 через веб-обновление обычно используется `.bin`.
- Для полной прошивки ESP32 используется `.merged.bin`.
- Для RP2040 используется `.uf2`.
